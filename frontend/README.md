# CloudFlow Frontend

CloudFlow 是一个可视化网页自动化工作流前端原型，基于 Vite、React 和 React Flow 构建。

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
